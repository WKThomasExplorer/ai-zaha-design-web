# AI Zaha Home Design - 项目规范

## 1. Problem Statement（问题陈述）

欧美自建房业主面临当地设计师资源稀缺、成本高昂的困境，在正式找设计师前缺乏快速验证设计方案的有效工具。同时，小型装修公司设计师急需快速生成外立面效果图以提升客户沟通效率，但现有方案要么成本高、要么质量差、要么操作复杂。

**核心痛点**：用户需要一个**零门槛、低成本、快速获得可用外立面效果图**的解决方案。

---

## 2. Proposed Solution（方案描述）

### 产品定位
一款面向欧美市场的**建筑外立面 AI 设计工具**，帮助用户：
- 欧美自建房业主：**先自行验证设计想法**，再对接施工队
- 装修公司设计师：**快速生成效果图**供客户确认

### 用户交互流程（分步确认）

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   [上传外立面图]  →  [选择风格/描述]  →  [发送]            │
│                                                             │
│   [效果图生成中...]                                        │
│                                                             │
│   [查看效果图]  ←──  [确认/重新生成]                       │
│       ↓                                                     │
│   [用户确认]  →  [爆炸图生成中...]                         │
│                                                             │
│   [效果图 + 爆炸图]  →  [下载结果]                         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

> **流程说明**：效果图生成后需要用户确认，确认后再生成爆炸图。

### 核心功能

| 功能 | 描述 |
|------|------|
| **图片上传** | 用户上传 1 张建筑外立面图（单图，JPG/PNG，最小 800px） |
| **风格模板** | 50 个预设风格选项，分为 5 大类（现代住宅、商业街区、工业改造、乡村民宿、前卫设计） |
| **需求描述** | 用户可选择模板或自定义描述想要的装修效果 |
| **效果图生成** | 直接生成装修效果图（无需确认步骤） |
| **爆炸图生成** | 基于效果图 AI 分析 + 用户描述，生成标注物料的爆炸图 |
| **重试机制** | 不满意可重新生成（整组重新生成） |
| **下载功能** | 用户可一键下载效果图、爆炸图和材料清单 CSV（完整包） |
| **博客页面** | 立面装修小贴士博客，包含文章列表和详情页 |
| **用户注册** | 用户可注册新账号（用户名+密码） |
| **用户登录** | 用户可登录已有账号 |
| **登录状态** | 登录后显示用户名，点击可登出 |

### 差异化亮点
- **爆炸图**：展示每一层使用的物料（红色陶土瓦、米色真石漆等），让用户知道"需要买什么"
- **快速生成**：选择模板后直接触发生成，无需 AI 确认步骤
- **双重校验**：爆炸图由 AI 分析效果图 + 用户描述共同生成

## 3. Technical Constraints（技术约束）

### 技术栈

| 层级 | 技术选型 |
|------|----------|
| **核心框架** | Next.js 16 (App Router) |
| **语言** | TypeScript 5 |
| **运行时** | Node.js 24 |
| **UI 框架** | React 19 |
| **样式** | Tailwind CSS 4 |
| **组件库** | shadcn/ui (Radix UI) |
| **图标** | Lucide React |
| **状态管理** | React Context API (ChatContext) |
| **包管理器** | pnpm（强制） |
| **开发工具** | Coze CLI |
| **代码规范** | Airbnb Style Guide |

### AI 集成
- **AI 能力**：全部使用 Coze 平台自带能力
- **生图方式**：分两次生成（效果图 → 爆炸图）
- **爆炸图生成逻辑**：AI 分析效果图 + 用户描述双重校验
- **材料分析**：爆炸图生成后，使用 Coze Vision API (doubao-seed-1-6-vision-250815) 自动识别图中的材料并填充到表格中

### API Routes

| 路由 | 方法 | 描述 |
|------|------|------|
| `/api/generate` | POST | 生成效果图和爆炸图 |
| `/api/analyze` | POST | 分析爆炸图，提取材料信息 |
| `/api/blog/generate` | GET | 生成博客文章内容 |
| `/api/auth/register` | POST | 用户注册 |
| `/api/auth/login` | POST | 用户登录 |
| `/api/auth/me` | GET | 获取当前登录用户信息 |

### 部署要求
- 服务端口：**5000**
- 环境：Coze 沙箱环境

## 4. Non-goals（明确不做的事）

以下内容**不在 MVP 范围内**：

| 不做的事 | 原因 |
|----------|------|
| ~~数据持久化/收藏功能~~ | 用完即走，不保存历史 |
| ~~分享链接功能~~ | 不需要，可直接截图分享 |
| ~~订阅/付费系统~~ | MVP 阶段免费，后期再考虑商业模式 |
| ~~多图片上传~~ | 每次只处理 1 张图 |
| ~~局部调整功能~~ | 不满意就整组重生成 |
| ~~材料清单导出~~ | 爆炸图展示即可 |
| ~~SEO 优化~~ | 独立工具，不依赖搜索引擎 |
| ~~数据统计分析~~ | MVP 阶段不关注数据 |
| ~~隐私政策/GDPR 合规~~ | 后期再加 |
| ~~小程序/嵌入 Widget~~ | 只做独立网站 |

## 7. 组件设计

### 页面状态流

```
IDLE → UPLOADED → GENERATING_EFFECT → EFFECT_READY → GENERATING_EXPLOSION → COMPLETED
  ↓         ↓              ↓                ↓               ↓              ↓
  └─────────┴──────────────┴────────────────┴───────────────┴──────────────┘
                                    （效果图确认后可重新生成）
```

> **分步说明**：
> 1. `GENERATING_EFFECT`：生成效果图
> 2. `EFFECT_READY`：用户确认效果图（可重新生成）
> 3. `GENERATING_EXPLOSION`：基于效果图生成爆炸图

### 核心组件

1. **ImageUploader** - 图片上传
   - 拖拽/点击上传
   - 图片预览
   - 格式校验（JPG/PNG）
   - 尺寸校验（最小 800px）

2. **StyleGallery** - 风格相册
   - 预设风格选项展示
   - 点击选择参考风格

3. **DescriptionInput** - 描述输入
   - 多行文本输入
   - 字数限制提示
   - 占位符引导

4. **AIConfirm** - AI 确认
   - 显示 AI 理解的摘要
   - 确认/修改按钮

5. **ResultViewer** - 结果查看
   - 效果图展示
   - 爆炸图展示
   - 下载按钮
   - 重新生成按钮

## 9. Zaha Hadid 设计风格指南

### 视觉特征
- **流动曲线**：大量使用有机曲线，避免硬直角
- **未来感**：简洁但富有科技感
- **有机几何**：建筑感的几何形态

### 配色方案
- 主色：深蓝灰 (#1a1f36) 或 深紫灰 (#2d2a4a)
- 强调色：渐变蓝绿 (#00d4aa → #0077ff)
- 背景：近白色 (#fafbfc) 或 深灰黑 (#0d0f14)
- 文字：深灰 (#2c3e50) 或 浅灰白 (#e8eaed)

### 字体选择
- 标题：几何感强的无衬线字体（如 Inter, Montserrat）
- 正文：清晰易读的无衬线字体

### 动效风格
- 柔和的渐变过渡
- 轻微的弹性动效
- 流线型的加载动画
